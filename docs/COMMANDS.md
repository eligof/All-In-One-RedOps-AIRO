# AIRO Command Index

This is a concise index of commands grouped by module. Use `airo help` and `airo modules` for live introspection.

## Network
- `netscan`, `portscan`, `udpscan`, `alivehosts`
- `dnscan`, `safescan`, `lhost`, `myip`
- `tracer`, `whoislookup`, `dnsdump`, `cidrcalc`

## Web
- `webscan`, `dirscan`, `fuzzurl`, `sqlcheck`, `xsscheck`
- `takeover`, `wpscan`, `joomscan`, `sslscan`, `headerscan`
- `httpxprobe`, `wayback`, `katana`, `nuclei`

## System
- `sysenum`, `sudofind`, `capfind`, `cronfind`
- `procmon`, `libfind`, `serviceenum`, `userenum`

## Privesc
- `lpe`, `wpe`, `sudoexploit`, `kernelcheck`
- `winprivesc`, `linprivesc`, `getpeas`

## Cloud / Containers
- `awscheck`, `azcheck`, `gcpcheck`
- `s3scan`, `ec2scan`
- `dockerscan`, `kubescan`, `containerbreak`

## Active Directory
- `adusers`, `adgroups`, `admachines`
- `bloodhound`, `kerberoast`, `asreproast`
- `goldenticket`, `silverticket`, `passpol`, `gpppass`

## Wireless
- `wifiscan`, `wifiattack`, `bluescan`, `blueattack`
- `wpscrack`, `handshake`, `pmkidattack`, `rfscan`

## Mobile / IoT
- `apkanalyze`, `apkdecompile`, `ipascan`, `androidscan`
- `iotscan`, `firmwareextract`, `bleenum`

## OSINT
- `emailosint`, `userosint`, `phoneosint`, `domainosint`
- `breachcheck`, `leaksearch`, `metadata`, `imageosint`

## Automation
- `reconall`, `vulnscan`, `reportgen`
- `findings`, `evidence`, `timertrack`, `notify`

## Utilities
- `urldecode`, `urlencode`, `base64d`, `base64e`
- `hexdump`, `filetype`, `calccidr`
- `shodanscan`, `censysscan`, `fofascan`
